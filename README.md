# mixgen
yt sound download and editing tool cli

## Install

### Linux / macOS / FreeBSD
```sh
curl -fsSL https://github.com/DsChauhan08/mixgen/releases/latest/download/install.sh | bash
# ensure ~/.local/bin (or PREFIX/bin) is on PATH, then:
mixgen
```

### Windows (PowerShell)
```powershell
.\install.ps1
# open a new terminal and run:
mixgen
```
